Edit/Delete messeges/notes in Chatter/Discuss

Video Tutorial

This module is allow user to edit/delete messages/notes in Chatter/Discuss.

Prerequisites before Installation

No prerequisites required before installation.

Configuration

  • Go to Settings
  • Select/Create User
  • Configure Chatter/Discuss Settings

Once you have checked "Allow to edit messages/notes" checkbox user can now edit messages/notes and this same applicable for delete messages/notes.

Edit Messages/Notes From Chatter/Discuss

  • User can see an edit icon button on notes.
  • On click of the above button it opens a wizard where user can edit note from message input box.
  • As you can see edited message also updated into chatter.
  • User can also track the notes history from History Tab.
  • User can also edit message icon button from Discuss's Channels also.

Delete Messages/Notes From Chatter/Discuss

  • User can also see a delete icon button on messages/ notes.
  • On click of the above button it opens a "confirm popup".
  • If you clicked on "OK" from confirm popup message/note will be deleted.
